# SchoolProject XMLファイルをPythonコードに変換するツールを作る  

* **使用技術**  

|    tool      |   version    |
|:------------:|:------------:|
|    Python    |    3.6.4     |
|    pip       |    9.0.2     |
|  xmltodict   |    0.11.0     |

* **仕様方法**  
 1. pythonの３系をinstallしてください(pipが使えればOK)　
 2. pip install xmltodict  
[How to use xmltodict](https://github.com/martinblech/xmltodict)
